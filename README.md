## Flexbox 

**Flexbox** — одномерная система компоновки для размещения элементов в строку 
или столбец.

Основные концепции:

* **Flex-контейнер** — родительский элемент с display: flex
* **Flex-элементы** — дочерние элементы внутри контейнера

Добавлять базовые стили можно в файле CSS


### Свойство flex-direction

По умолчанию используется свойство **flex-direction: row**.

Другие возможные значения для **flex-direction**:

* row — элементы в строку (→)
* row-reverse — элементы в строку справа налево (←)
* column — элементы в столбец (↓)
* column-reverse — элементы в столбец снизу вверх (↑)


### Свойство justify-content

**justify-content** — выравнивание по главной оси

Другие возможные значения для  **justify-content**:

* **flex-start** — в начале (по умолчанию)
* **flex-end** — в конце
* **center** — по центру
* **space-between** — равномерно, первый и последний у краёв
* **space-around** — равномерно, с отступами по краям
* **space-evenly** — равномерно, одинаковые отступы везде


### align-items

**align-items** — выравнивание по поперечной оси

Другие возможные значения для **justify-content**:

* **stretch** — растягивает элементы (по умолчанию)
* **flex-start** — в начале
* **flex-end** — в конце
* **center** — по центру
* **baseline** — по базовой линии текста


### flex-wrap

**flex-wrap** — перенос элементов

Возможные значения для переноса элементов:

* **nowrap** — без переноса (по умолчанию)
* **wrap** — перенос на новую строку
* **wrap-reverse** — перенос в обратном направлении


### Другие возможности

**flex-grow** — растягивание элемента

**flex-basis** — базовый размер (ширина) элемента

**flex-shrink** — сжатие элемента

**flex** — краткая запись

**align-self** — индивидуальное выравнивание
